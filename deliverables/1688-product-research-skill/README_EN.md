# 1688 Product Research Skill

A skill for researching products on 1688.com using Playwright browser automation.

## Files

- `src/research.py` - Main research script
- `1688cookie.json.example` - Cookie file template
- `requirements.txt` - Python dependencies
- `output/` - Research results output directory

## Usage

1. Copy `1688cookie.json.example` to `1688cookie.json` and fill in your cookies
2. Run: `python -m src.research`

## Note

The actual cookie file is not included. You need to export your own cookies from 1688.com.
